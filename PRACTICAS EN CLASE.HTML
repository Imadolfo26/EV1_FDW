Actividades realizadas por el equipo

Actividad 1: Curriculum Vitae

En esta actividad se elaboró un Curriculum Vitae utilizando HTML5 y CSS. El objetivo fue aprender a estructurar información personal y profesional de manera clara y atractiva, aplicando estilos que realzan la presentación.

Actividad 2: Bandera de México

Esta actividad consistió en crear la Bandera de México utilizando HTML5 y CSS. Se aplicaron efectos de interacción al pasar el cursor, lo que mejoró la experiencia del usuario al hacer la bandera interactiva y visualmente atractiva.

Actividad 3: Pronóstico del Clima

En esta práctica, se desarrolló una página web que muestra un pronóstico del clima para San Nicolás de los Garza, empleando HTML5 y el framework de Bootstrap. Se trabajó en la integración de elementos visuales y funcionales, como tarjetas informativas que presentan datos climáticos de manera clara y accesible.

La actividad consistió en crear un formulario de registro estilizado, utilizando HTML5 y CSS Se enfocó en la recolección de datos del usuario, implementando validaciones.
